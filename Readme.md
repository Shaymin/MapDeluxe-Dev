MapDeluxe
==========

相关信息
----------
开放源代码的《星之卡比：梦之泉的噩梦》地图编辑器   
原作者：百度星之卡比吧 wwylele(http://tieba.baidu.com/i/sys/jump?un=wwylele)   
语言：C++ (MFC)   
开发环境：Visual Studio 2008(wwylele)/2010(冰空亅花束)   
运行环境：Windows XP/Vista/7/8? 32bit   

快捷键
----------
WASD控制选择的光标移动（红）
Z切换图像/判定
X切换掀开/不掀开
